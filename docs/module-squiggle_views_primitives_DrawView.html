Module: squiggle/views/primitives/DrawView

DrawView is the base class for all View's that have 'drawable' properties
Properties:
Name Type Description
strokeWeight Number The stroke weight to use when drawing.
strokeColor String The stroke color to use then drawing as a CSS rgba string
strokeCap Number The stroke cap style to use when drawing.
strokeJoin Number The stroke join style to use when drawing.
fill Boolean Whether or not to fill when drawing.
fillColor String The color to use when 'filling' as a CSS rgba string
closed Boolean Whether or not the Object is 'closed' (ie. a path can be open or closed)
jerkiness Number The amount of randomness applied when drawing (default 0.5)
defaultJerkinessBump Number jerkiness value that is set in the jerkIt() effect if no value is provided
defaultJerkinessBumpIncrease Number amount that gets added/substracted from the jerkiness on each draw() step after jerkIt() has been called
