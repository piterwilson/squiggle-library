Module: squiggle/views/ui/Button

The Button class is a View that holds a Word instance and a Line instance, and adds basic interactive behaviors : click, hover and down
Properties:
Name Type Description
padding Number Padding inside the button
width Number width in pixels of the button
height Number height in pixels of the button
autoAdjustSize Boolean Whether or not to auto adjust the width and height of the Button when the text is set
showUnderline Boolean Whether or not to show an underline on roll over
