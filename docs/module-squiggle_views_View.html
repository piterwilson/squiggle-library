Module: squiggle/views/View

A View class represents anything that will be rendered on screen.

Properties:
Name Type Description
x Number The x position of the View
y Number The y position of the View
offsetX Number A value that is meant to contain the x position of the parent of the View
offsetY Number A value that is meant to contain the y position of the parent of the View
hidden Boolean Whether or not to draw the view on screen
userInteractionEnabled Boolean Whether or not the View should be checked for p5.js interactive function calls (mouseMoved(), mousePressed() etc...)
