DashedLine is a View that renders a dashed line
Properties:
Name Type Description
spacing Number The spacing between the dashes in the dashed line rendered.
steps Number The number to steps (p5renderer frames) to wait before switching the dashes to spaces and viceversa. ie the 'speed' of the line
