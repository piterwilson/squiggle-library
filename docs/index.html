Squiggle

Description

The Squiggle library is a set of classes that are ready to use to implement animation drawing applications that output animated .gif's.

Squiggle is made using p5.js

Dependencies

The Squiggle library requires:

  • require.js
  • jquery
  • Underscore
  • Backbone

Acknowledgments

  • Squiggle uses p5.js under the hood for its drawing routines.
  • Squiggle uses gif.js to export .gif's.
  • Squiggle uses randomColor to generate nice random colors.
