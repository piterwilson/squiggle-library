Module: squiggle/views/text/Letter

Letter a class to encapsulate the concept of an animated typographic character. At its core, it is a View that draws a set of lines using a 'blueprint'.

Properties:
Name Type Description
blueprint Array An Array of abstract positions that describe the lines used to draw the Letter
widthFactor Number A factor used to determine the Letter's width
fontSize Number Size in pixels for the Letter
